# Source

This folder contains the source code for NumpadOS.

## Files

- `Launcher.ahk` - Entry point of the application
- `Main.ahk` - Main program logic
- `Apps.ahk` - Application launcher module
- `Windows.ahk` - Window management module
- `Helpers.ahk` - Shared helper functions
- `Toast.ahk` - Notification system
- `Config.ini` - User configuration
