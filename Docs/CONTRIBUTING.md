# Contributing to NumpadOS

Thank you for your interest in contributing!

## How to contribute

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Submit a Pull Request.

## Coding Style

- Use AutoHotkey v2
- Keep code modular
- Comment important functions
- Write clear commit messages

Thank you for helping improve NumpadOS!
