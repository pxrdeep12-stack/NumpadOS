# NumpadOS v1.0.0 Specification

## Goal

Turn an unused numpad into a productivity launcher for Windows using AutoHotkey v2.

## Features

- Launch applications
- Open websites
- Open folders
- Run commands
- Window management
- Toast notifications
- Config.ini configuration

## Not Included

- GUI
- Plugin system
- Themes
- Auto updater
- Cloud sync

## Target

A lightweight launcher that starts instantly and is fully controlled from the numpad.
