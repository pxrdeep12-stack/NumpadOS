# Changelog

## 🚀 NumpadOS v2.0.0

This release is a complete redesign of NumpadOS, introducing a modern graphical interface, a modular architecture, and a much easier configuration experience.

---

## ✨ Added

- Brand new graphical Configurator
- Modern shortcut creation wizard
- Support for Applications (.exe)
- Support for Folders
- Support for Websites
- Support for Keyboard Shortcuts
- Smart Launch system
- JSON-based configuration
- Professional system tray menu
- Launch at Windows startup option
- Configuration validation
- Automatic logging
- Import and Export configuration
- Search support
- Automatic icon detection and caching
- Window activation for supported applications
- Improved configuration management

---

## 🔄 Changed

- Complete project architecture redesigned
- Migrated from INI configuration to JSON
- Reorganized project into modular libraries
- Improved launcher reliability
- Improved configuration workflow
- Simplified setup process
- Updated README with screenshots and installation guide
- Cleaner and more maintainable codebase

---

## ⚡ Improved

- Faster startup
- Better error handling
- Better configuration validation
- Improved Smart Launch logic
- Better logging system
- More consistent user interface
- Easier future feature expansion
